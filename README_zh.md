# 长辈桌面 (Elder Desktop)

[简体中文] | [English](README.md)

这是一个专门为老年人设计的手机主屏幕（启动器）。本项目为预发布版本，可能存在漏洞；如有任何问题，请通过 Issues 部分进行反馈。

系统要求：
本软件支持 Android 6.0 及以上版本。

## 广告 SDK 限制
本项目严格开源且非商业化。**严禁**包含任何广告 SDK（包括但不限于 AdMob、Unity Ads、Pangle 等）。软件包含自动化运行时检查以执行此政策。

## 功能特性

[X] 多布局桌面

[X] 语音助手按钮

[X] 兼容绝大多数设备 (≈99.6%)

[X] 支持 Android 6.0 及更高版本，目前不支持 iOS 或 HarmonyOS NEXT。

[X] 支持通过“设置”编辑桌面并移动图标和组件。

[X] 使用默认布局加载时，主屏幕显示所选应用的设置默认处于选中状态。

[X] 无论处于深色还是浅色模式，如果图标按钮的背景色接近白色，应用标题应显示为黑色；如果接近黑色，则应显示为白色。

[X] 支持 1 x 2 布局和单列滚动。

## 警告

本软件尚未通过 Google Play 授权验证，未来安装可能会被拦截；因此，中国大陆制造的大多数手机仍不受影响。
