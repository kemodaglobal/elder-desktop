# 长辈桌面附加开源使用规则（非许可证替代方案）

[简体中文] | [English](RULES.md)

本项目根据 Apache License 2.0 获得许可。
以下内容构成适用于所有分叉、修改和再分发活动的**附加使用条款**。
目标是确保新旧 Android 手机（尤其是 60 至 80 岁老年人使用的手机）之间的一致用户体验。

## 规则

1. **禁止添加广告**：严禁出于任何原因在桌面的任何区域添加广告。这包括嵌入第三方广告 SDK。软件采用运行时检查来验证合规性。
2. **禁止随意修改 minSdk 和 targetSdk**：
   - 原始 minSdk / targetSdk 值请参见 `build.gradle.kts`。
   - 请勿降低 `minSdk`，也不要随意更改 `targetSdk` 以规避系统行为（如权限或后台限制）。
   - 本项目使用 Gradle 版本 9.6.1、Android Gradle 插件版本 9.3.0 和 targetSdk 37；因此，开发需要 Android Studio Quail 2 或更高版本。
3. **披露兼容性信息的义务**：如果必须调整本开源软件的 `targetSdk`，必须在 `targetSdk` 文档中详细说明对新设备的影响。
4. **“安心锁”的独立安全性**：如果为设置保护实现了指纹识别，则不得使用为系统屏幕锁定注册的指纹。它应为家庭成员维护一条独立的身份验证路径。
5. **手机独占性**：本软件专为手机和折叠屏设备设计和优化。明确排除对智能手表 (Wear OS) 的支持，以保持 UI 简洁性和针对老年人的易用性。
