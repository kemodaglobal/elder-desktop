# Elder Desktop

[English] | [简体中文](README_zh.md)

This is a mobile home screen designed specifically for the elderly. This project is a pre-release version and may contain bugs; please report any issues via the Issues section.

System Requirements:

This software supports Android 6.0 or later.

## Advertising SDK Restriction
This project is strictly open-source and non-commercial. The inclusion of any advertising SDK (including but not limited to AdMob, Unity Ads, Pangle, etc.) is **strictly prohibited**. The software includes automated runtime checks to enforce this policy.

## Features

[X] Multi-layout desktop

[X] Voice assistant button

[X] Compatible with the vast majority of devices (≈99.6%)

[X] Supports Android 6.0 and later versions, but does not currently support iOS or HarmonyOS NEXT.

[X] Supports editing the desktop and moving icons and widgets within the desktop via Settings.

[X] If loading with the default layout, the setting to display selected apps on the Home screen is checked by default.

[X] Regardless of whether dark mode or light mode is active, the app title should be black if the icon button's background color is close to white, and white if it is close to black.

[X] 1 x 2 layout support and single-column scrolling

## Warning

This software has not undergone Google Play authorization verification, and its installation will be blocked in the future; therefore, most mobile phones manufactured in mainland China remain unaffected.
