# Elder Desktop

This is a mobile home screen designed specifically for the elderly. This project is a pre-release version and may contain bugs; please report any issues via the Issues section.

System Requirements:

This software supports Android 6.0 or later.

## Advertising SDK Restriction
This project is strictly open-source and non-commercial. The inclusion of any advertising SDK (including but not limited to AdMob, Unity Ads, Pangle, etc.) is **strictly prohibited**. The software includes automated runtime checks to enforce this policy.
